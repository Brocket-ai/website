# Website

Landing page / sitio de marketing de Brocket.

## Stack

- **Framework**: Next.js 16
- **Styling**: TailwindCSS
- **Components**: shadcn/ui

## Estructura

```
website/
├── app/              → App router (Next.js)
├── components/       → UI components
├── public/           → Assets estáticos
└── styles/           → CSS global
```

## Comandos

```bash
npm run dev     # Desarrollo
npm run build   # Build
```

## Notas

Este repo es independiente del producto principal. Es solo marketing/landing.
